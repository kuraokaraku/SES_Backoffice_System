## 1. ログイン・ログアウト

### ログイン

1. ブラウザで https://itfl-backoffice.com にアクセス
2. ユーザー名とパスワードを入力
3. 「ログイン」ボタンをクリック

### ログアウト

- 画面右上のログアウトボタンをクリック

---

## 2. ダッシュボード（トップ画面）

ログイン後に最初に表示される画面です。業務の全体状況がひと目でわかります。

### 表示される情報

| 項目 | 意味 |
|------|------|
| 未回収タイムシート | まだ届いていない勤怠報告の件数 |
| 未送付請求書 | 作成済みだがまだ送っていない請求書の件数 |
| 未入金 | 請求済みだがまだ入金がない件数 |
| 契約期限切れ間近 | もうすぐ契約が終了するアサインメントの件数 |
| 停滞中の案件 | 7日以上動きがない営業案件の件数 |

### 売上推移グラフ

過去6か月の売上・原価・粗利の推移が表示されます。

### 月の切り替え

画面上部で対象月を切り替えると、その月のデータが表示されます。

## 3. 人員管理（パーティ一覧）

**メニュー**: パーティ一覧

フリーランス・ビジネスパートナー・社員など、業務に関わる人や会社を管理する画面です。

### 新規登録

1. 「新規登録」ボタンをクリック
2. 種別を選択（個人 or 法人）
3. 必要情報を入力
   - **個人の場合**: 氏名、メールアドレス、電話番号、ワーカー種別（フリーランス / BP / 社員 / 業務委託）
   - **法人の場合**: 会社名、住所、電話番号、インボイス登録の有無
4. アサインメント（案件情報）も同時に登録可能
5. 「保存」をクリック

### 一覧画面でできること

- 名前やプロジェクト名で検索
- 各人のアサインメント詳細へ移動

## 4. アサインメント（案件・契約管理）

**メニュー**: パーティ一覧 → 対象者をクリック

1つのアサインメントは「誰が」「どの案件で」「どの会社の間で」働いているか、という情報です。

### アサインメント詳細画面

以下の情報が表示されます：

- **基本情報**: ワーカー名、プロジェクト名、営業担当、上流（顧客）、下流（発注先）
- **契約条件**: 単価、精算時間の上限・下限、超過単価、控除単価、支払いサイト

### 契約条件の見方

| 用語 | 意味 |
|------|------|
| 単価 | 1か月あたりの基本金額 |
| 下限時間 | この時間を下回ると控除（減額）が発生する |
| 上限時間 | この時間を超えると超過（追加）が発生する |
| 控除単価 | 下限を下回った場合の1時間あたりの減額金額 |
| 超過単価 | 上限を超えた場合の1時間あたりの追加金額 |
| 精算単位 | 15分 / 30分 / 60分 のいずれか |
| 支払いサイト | 請求後、何日以内に支払われるか（例: 30日、60日） |

### 契約の延長

1. アサインメント詳細画面で「契約延長」ボタンをクリック
2. 新しい契約終了日を入力（必要に応じて単価等も変更可能）
3. 「保存」をクリック

### アサインメントの有効/無効

- 案件が終了した場合、「無効にする」で非アクティブにできます
- 再開する場合は「有効にする」で戻せます

## 5. 勤怠管理（タイムシート）

**メニュー**: タイムシート

毎月のワーカーの勤務時間報告を管理する画面です。

### タイムシート一覧

対象月ごとに、各ワーカーのタイムシート提出状況が一覧で表示されます。

| ステータス | 意味 |
|------------|------|
| 未受領 | まだ提出されていない |
| 受領済 | ファイルを受け取った |
| 処理済 | 内容を確認して処理が完了した |

### タイムシートのアップロード

1. 「アップロード」ボタンをクリック
2. 対象のアサインメントと対象月を選択
3. ファイル（PDF または Excel）を選択
4. 「アップロード」をクリック

アップロード後、システムがファイルから **実稼働時間を自動で読み取り** ます。

### タイムシートから請求書を作成

1. タイムシート詳細画面を開く
2. 実稼働時間が正しいことを確認
3. 「請求書を作成」ボタンをクリック
4. 契約条件に基づいて請求書が自動作成されます

## 6. 請求書管理

**メニュー**: 請求書

顧客への請求書を作成・管理する画面です。

### 請求書のステータス

| ステータス | 意味 |
|------------|------|
| 下書き | 作成中。まだ編集可能 |
| 確定 | 内容が確定した。編集不可 |
| 送付済 | 顧客に送付した |
| 取消 | キャンセルされた請求書 |

### 請求書の作成方法

請求書は主に2つの方法で作成します：

1. **タイムシートから自動作成**（推奨）
   - タイムシート画面で「請求書を作成」をクリック
   - 契約条件と実稼働時間から金額が自動計算される

2. **手動アップロード**
   - 請求書一覧の「アップロード」から直接ファイルをアップロード

### 請求書の確定

1. 請求書詳細画面で内容を確認
2. 「確定する」ボタンをクリック
3. 確定後は金額の変更ができなくなります

### Excel 出力

- 請求書詳細画面で「Excel出力」ボタンをクリック
- 会社の雛型テンプレートに沿った Excel ファイルがダウンロードされます

### 送付済みにする

- 顧客に請求書を送った後、「送付済みにする」をクリック

## 7. 入金管理（売掛）

**メニュー**: 入金管理

請求書に対する入金（お金を受け取ること）を記録する画面です。

### 入金の記録

1. 入金管理一覧から対象の請求書を選択
2. 「入金を記録」をクリック
3. 入金日・金額・メモを入力
4. 「保存」をクリック

### 入金状況の見方

| 状態 | 意味 |
|------|------|
| 未入金 | まだ入金がない |
| 一部入金 | 請求額の一部だけ入金された |
| 入金済 | 全額入金された |

## 8. 支払管理（買掛）

**メニュー**: 支払管理

フリーランスやパートナーへの支払い（お金を払うこと）を記録する画面です。

操作方法は入金管理と同じです。

### 支払いの記録

1. 支払管理一覧から対象の支払いを選択
2. 「支払いを記録」をクリック
3. 支払日・金額・メモを入力
4. 「保存」をクリック

## 9. 営業管理（セールスボード）

**メニュー**: 営業ボード

営業案件をカンバン形式（ふせんを動かすイメージ）で管理する画面です。

### ステータスの流れ

```
受領 → 対応中 → 提案済 → 結果待ち → 成約 or 失注
```

| ステータス | 意味 |
|------------|------|
| 受領 | 案件情報を受け取った段階 |
| 対応中 | 候補者の選定や書類作成を進めている |
| 提案済 | 顧客に候補者を提案した |
| 結果待ち | 面談後、顧客の回答を待っている |
| 成約 | 契約が決まった |
| 失注 | 見送りになった |

### 新規案件の登録

1. 「新規案件」ボタンをクリック
2. プロジェクト情報を入力（会社名、案件名、必要スキル、予算帯）
3. 候補者情報を入力（候補者名、担当者）
4. 「保存」をクリック

### 案件の進捗を動かす

1. 案件カードをクリックして詳細画面を開く
2. 「ステータスを変更」からステータスを選択

### 営業活動の記録

案件詳細画面で、日々の活動を記録できます：

- 電話、メール、メッセージ、面談、書類、面接、提案、その他
- 活動内容のメモも入力可能

### 注意マーク

- **停滞**: 7日以上活動がない案件に表示されます
- **期限超過**: 次のアクション予定日を過ぎた案件に表示されます

## 11. 月次処理

**メニュー**: 月次処理

毎月の業務進捗をアサインメントごとにまとめて管理する画面です。

### 月次処理の作成

1. 「一括作成」ボタンをクリック
2. 対象月を選択
3. アクティブなアサインメント分の月次レコードが自動作成されます

### 進捗の更新

各アサインメントについて、以下のステータスを更新できます：

- タイムシート: 未受領 / 受領済 / 処理済
- 請求書: 未作成 / 作成済 / 送付済
- 注文書: 未受領 / 受領済

## 12. ユーザー管理

**メニュー**: ユーザー管理（管理者のみ）

システムにログインするユーザーのアカウントを管理します。

### ユーザーの作成

1. 「新規作成」ボタンをクリック
2. ユーザー名、パスワード、権限を設定
3. 「保存」をクリック

### パスワードの変更

1. ユーザー一覧から対象ユーザーの「編集」をクリック
2. 新しいパスワードを入力
3. 「保存」をクリック

## 13. よくある質問

### Q. 金額が合わない

請求書の金額は契約条件（単価・上限時間・下限時間）と実稼働時間から自動計算されます。
金額がおかしい場合は、以下を確認してください：

1. タイムシートの実稼働時間は正しいか？
2. アサインメントの契約条件（単価・時間枠）は正しいか？

### Q. 請求書を修正したい

- **下書き状態**: 詳細画面から編集できます
- **確定済み**: 一度取消して、新しい請求書を作成してください

### Q. 契約条件を変更したい

アサインメント詳細画面の「編集」から変更できます。ただし、過去の請求書には影響しません。

### Q. パスワードを忘れた

管理者に連絡して、パスワードをリセットしてもらってください。
