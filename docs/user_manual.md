## 1. ログイン・ログアウト

### ログイン

1. ブラウザで https://itfl-backoffice.com にアクセス
2. ユーザー名とパスワードを入力
3. 「ログイン」ボタンをクリック

### ログアウト

- 画面右上のログアウトボタンをクリック

---

## 2. ダッシュボード（トップ画面）

ログイン後に最初に表示される画面です。業務の全体状況がひと目でわかります。

### 表示される情報

| 項目 | 意味 |
|------|------|
| 未回収タイムシート | まだ届いていない勤務表の件数 |
| 未送付請求書 | 作成済みだがまだ送っていない請求書の件数 |
| 未入金 | 請求済みだがまだ入金がない件数 |
| 契約終了30日以内 | もうすぐ契約が終了するアサインメントの件数 |
| 契約終了60日以内 | 60日以内に契約が終了するアサインメントの件数 |

### 月の切り替え

画面上部で対象月を切り替えると、その月のデータが表示されます。

## 3. 契約管理

**メニュー**: 契約管理

人材・案件・契約条件を管理する画面です。

### 3-1. 一覧画面

- 稼働中のアサインメントを一覧表示
- 名前・プロジェクト名で検索可能
- 「稼働中」「終了」の切り替えフィルタあり

### 3-2. 新規登録

1. 「新規登録」ボタンをクリック
2. 以下の情報を入力：
   - **人材情報**: 氏名、種別（フリーランス / BP / 社員 / 業務委託）、メール、電話
   - **案件情報**: プロジェクト名、営業担当
   - **上位会社**: 発注元の会社・担当者
   - **下位会社**: 発注先の会社・担当者
   - **契約条件**: 単価、精算幅、超過/控除単価、支払サイト等
3. 「保存」をクリック

#### 契約書からAI自動入力（便利機能）

契約書ファイル（Excel/PDF）をアップロードするだけで、フォームの各項目を自動入力できます。

1. 「契約書からAI自動入力」ボタンをクリック
2. 契約書ファイル（ExcelまたはPDF）を選択
3. 解析が完了すると画面が2分割で表示される
   - **左側**: 契約書のプレビュー（抽出した項目が黄色でハイライト）
   - **右側**: AIが読み取った内容（編集可能）
4. 内容を確認・修正して「フォームに反映」をクリック
5. 登録フォームに自動入力されるので、確認して「保存」

> **ポイント**: 「信頼度」が表示されます。100%に近いほど正確に読み取れています。不明な項目は空欄になるので手動で補完してください。

### 3-3. アサインメント詳細

案件の詳細情報を確認する画面です。

**表示内容**:
- 人材情報・案件情報
- 契約条件（単価・精算幅・超過/控除単価・支払サイト）
- 上位・下位会社の情報

**できること**:
- 「編集」で情報を変更
- 「見積書」で見積書Excelをダウンロード
- 「稼働終了」で非アクティブにする

### 3-4. 契約条件の用語

| 用語 | 意味 |
|------|------|
| 単価 | 1か月あたりの基本金額（税抜） |
| 精算下限時間 | この時間を下回ると控除（減額）が発生 |
| 精算上限時間 | この時間を超えると超過（追加）が発生 |
| 控除単価 | 下限を下回った場合の1時間あたりの減額金額 |
| 超過単価 | 上限を超えた場合の1時間あたりの追加金額 |
| 精算単位 | 端数処理の単位（例: 30分単位で切り捨て） |
| 支払サイト | 請求後、何日以内に入金されるか（例: 40日） |

## 4. 勤務表回収

**メニュー**: 勤務表回収

毎月のワーカーの勤務時間報告を管理する画面です。

### 4-1. ダッシュボード

対象月ごとに、各ワーカーの勤務表提出状況が一覧で表示されます。

| ステータス | 意味 |
|------------|------|
| 未受領 | まだ提出されていない |
| 受領済 | ファイルを受け取った |
| 請求書生成済 | 請求書の作成まで完了した |

### 4-2. 勤務表のアップロード

1. 「アップロード」ボタンをクリック
2. 対象のアサインメントと対象月を選択
3. ファイル（PDF または Excel）を選択してアップロード

アップロード後、システムが自動で **実稼働時間・交通費を読み取り** ます。

### 4-3. 勤務表の確認・修正

1. 勤務表をクリックして詳細画面を開く
2. 自動読み取りした実稼働時間・交通費を確認
3. 誤りがあれば手動で修正して「保存」

### 4-4. 請求書の生成

1. 勤務表詳細画面で「請求書を作成」をクリック
2. 契約条件と実稼働時間をもとに請求書が自動作成されます

## 5. 請求書管理

**メニュー**: 請求書

顧客への請求書を管理する画面です。

### 5-1. ステータス

| ステータス | 意味 |
|------------|------|
| 下書き | 作成中。編集可能 |
| 確定 | 内容が確定した。編集不可 |
| 送付済 | 顧客に送付した |
| 取消 | キャンセルされた請求書 |

### 5-2. 請求書の確認・編集

1. 請求書一覧から対象をクリック
2. 詳細画面で金額・明細を確認
3. 必要に応じて手動で編集
4. 「再計算」ボタンで契約条件から金額を再計算することも可能

### 5-3. 請求書の確定

1. 詳細画面で「確定する」をクリック
2. 確定後は金額の変更不可

> **注意**: 確定後に修正が必要になった場合は「取消」を押してから新たに作り直してください。

### 5-4. Excel出力と送付

1. 詳細画面で「Excel出力」をクリック → 雛型テンプレートでダウンロード
2. 顧客に送付後、「送付済みにする」をクリック

## 6. 入金管理（売掛）

**メニュー**: 入金管理

請求書に対する入金を記録する画面です。

### 入金の記録

1. 一覧から対象の請求書を選択
2. 「入金を記録」をクリック
3. 入金日・金額・メモを入力して「保存」

### 入金状況の見方

| 状態 | 意味 |
|------|------|
| 未入金 | まだ入金がない |
| 一部入金 | 請求額の一部だけ入金された |
| 入金済 | 全額入金された |
| 期限超過 | 支払期日を過ぎても未入金 |

---

## 7. 支払管理（買掛）

**メニュー**: 支払管理

フリーランスやパートナーへの支払いを記録する画面です。操作方法は入金管理と同じです。

### 支払いの記録

1. 一覧から対象を選択
2. 「支払いを記録」をクリック
3. 支払日・金額・メモを入力して「保存」

---

## 8. ユーザー管理

**メニュー**: ユーザー管理（管理者のみ）

システムにログインするユーザーのアカウントを管理します。

### ユーザーの作成

1. 「新規作成」ボタンをクリック
2. ユーザー名・パスワード・権限を設定
3. 「保存」をクリック

### パスワードの変更

1. ユーザー一覧から対象の「編集」をクリック
2. 新しいパスワードを入力して「保存」

## 9. よくある質問

### Q. 金額が合わない

請求書の金額は契約条件と実稼働時間から自動計算されます。以下を確認してください：

1. 勤務表の実稼働時間は正しいか？
2. アサインメントの契約条件（単価・精算幅）は正しいか？
3. 請求書詳細画面で「再計算」を試してみてください

### Q. 請求書を修正したい

- **下書き状態**: 詳細画面から直接編集できます
- **確定済み**: 「取消」してから新しい請求書を作成してください

### Q. 契約条件を変更したい

アサインメント詳細の「編集」から変更できます。過去の請求書には影響しません。

### Q. AI自動入力の精度が低い

- テキストが埋め込まれたExcel/PDFで精度が高くなります（画像スキャンのPDFは非対応）
- 信頼度が低い項目は手動で確認・修正してください
- 抽出できなかった項目は空欄になります

### Q. パスワードを忘れた

管理者に連絡してパスワードをリセットしてもらってください。
